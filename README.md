# List of sellers and orders

## This project uses SQLite and Prisma to build a backend development environment (database)
<br />

### Install the dependencies

```bash
npm install
```
### Access with database

```bash
npx prisma db pull
```

### Populate the database

```bash
npx prisma db seed
```

### Open prisma interface

```bash
npx prisma studio
```

### Run backend

```bash
npm run dev
```
